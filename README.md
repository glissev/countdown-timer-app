# countdown-timer-app

A simple countdown timer web app hosted on GitHub Pages.

🔗 **Live demo:** https://glissev.github.io/countdown-timer-app/

## Features

- Set a countdown to any future date and time
- Displays total days, weeks, weekdays, and weekend days remaining
- Live countdown in days, hours, minutes, and seconds
- Saves your countdown across page refreshes using localStorage

## Hosting

This app is hosted on [GitHub Pages](https://pages.github.com/) and automatically deployed via GitHub Actions whenever changes are pushed to the `main` branch.
